The banner fragment can be used to announce promotions and/or updates on your store. It consists of a title, description and a call-to-action button. Furthermore, you can choose to fix the banner in a certain position on your store and allow visitors to close the banner.

| Option | Description |
| :--- | :--- |
| Title | Title |
| Description | Description (can be disabled) |
| Button | Call-to-action button (can be disabled) |
| Closable | Whether banner can be closed by visitor |
| Fixed | Whether banner has fixed position on the screen |
