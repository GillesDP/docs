Grids are a great way to organize important features about your store and product. It's easy to read and can be used to highlight key selling points. Each grid element consists of an icon, title, and description. You can also have an element link to some page or external site.

| Option | Description |
| :--- | :--- |
| Containerize | Whether each grid element needs its own container with background |
| Align | How content should be aligned within each grid item |
| Elements | List of grid elements |
